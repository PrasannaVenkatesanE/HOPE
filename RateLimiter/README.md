# Day 16 - Token Bucket Rate Limiter

This example demonstrates a simple Token Bucket rate limiter implemented in Java.

## Files

- `TokenBucketRateLimiter.java`: Token bucket implementation with refill logic.
- `TokenBucketDemo.java`: Demo program that consumes tokens and shows request acceptance/rejection.

## How to run

From the `Day-16` directory:

```bash
javac TokenBucketRateLimiter.java TokenBucketDemo.java
java TokenBucketDemo
```
